# TikTok Demo 🎬

纯粹、极简、无后端的 TikTok 视频 / 图片下载工具。

## 🔧 技术原理

- **客户端解析**：100% 纯前端实现，通过 Fetch API 直接调用公开解析接口。
- **无水印处理**：实时解析 TikTok 元数据，提取无水印高清视频 (HD) 及原始图片资源。
- **隐私安全**：不设后端服务器，不存储任何用户链接或下载记录，所有操作均在浏览器完成。
